# Nelio92.github.io

Static GitHub Pages portfolio site for Lionel Wilfried Wandji.

## Files

- `index.html` - site structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - simple reveal and active-navigation behavior

## Publish Options

### Personal site repository

Create or use the GitHub repository `Nelio92.github.io` and place these files at the repository root.

### Project site repository

You can also host this from another repository by enabling GitHub Pages and publishing from the branch/folder that contains these files.
